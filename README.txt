Tips voor bijdragers:
    - Werk in de virtual environment uit .venv zodat we allemaal in vergelijkbare
    omgevingen werken. Start de virtual omgeving door "source .venv/bin/activate"
    te runnen en installeer de benodigde packages door middel van
    "pip install -r requirements.txt". Stop de virtuele omgeving met "deactivate"
    - Hou je zo veel mogelijk aan de pep8 python style guide. black is een goede
    tool voor python reformatting of configureer een autoformatter in je editor.
