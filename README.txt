Tips voor bijdragers:
    - Werk in de virtual environment uit .venv zodat we allemaal in vergelijkbare
    omgevingen werken. Maak de virtuele omgeving aan door "python -m venv .".
    Start de virtual omgeving door "source venv/bin/activate" te runnen en
    installeer de benodigde packages door middel van "pip install -r requirements.txt".
    Stop de virtuele omgeving met "deactivate"
    - Hou je zo veel mogelijk aan de pep8 python style guide. black is een goede
    tool voor python reformatting of configureer een autoformatter in je editor.
    - Voor grote stukken code maak een aparte branch aan, kleine patches van
    maximaal 10 of 15 lijnen kunnen wel direct naar de main branch.
    - Documenteer de functies die je schrijft. Zet bij de functiedeclaratie 
    typedefinities of docstrings om te documenteren wat voor datatypes bij de
    argumenten en uitvoer horen.
